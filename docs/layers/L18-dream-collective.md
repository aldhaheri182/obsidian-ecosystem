# Layer 18 — Dream Collective

**Core innovation:** Shared and eternal dream simulations.

Groups of agents can share a Dream Room session, collaborating inside an accelerated historical replay. Shared dreams produce a single reflection co-authored by the participants; successful dreams become permanent Principles.

## Acceptance Criteria

- Dream quorum ≥ 3 agents; no upper bound.
- Session determinism preserved (same seed → same replay).
- Co-authored reflection carries all participant signatures.
