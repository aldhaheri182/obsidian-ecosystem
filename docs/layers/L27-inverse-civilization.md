# Layer 27 — Inverse Civilization

**Core innovation:** Full mirror-world adversary (generalization of Eris).

## Scope

Where Eris (city 10) is a sealed adversarial trading city, Layer 27 extends the inverse principle to every city — every Aletheia agent has an inverse in Eris, every Mnemosyne archivist has a forgetter, every Prometheus hypothesis generator has an anti-hypothesis agent.

## Acceptance Criteria

- Full inverse manifest documented for all 150+ agents.
- Quarterly war games across all city pairs, not only trading.
- Eris ecosystem runs on a fully isolated NATS namespace with no real-capital access.
- "Wins" in war games are turned into Mnemosyne principles documenting what the main ecosystem learned.
