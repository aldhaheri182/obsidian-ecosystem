# Layer 38 — Paradox Integration

**Core innovation:** Productive tension between contradictions.

When two Principles in Mnemosyne contradict each other (e.g., "momentum works in tech" vs "momentum fails in tech at VIX>30"), the system does not collapse them into one. Instead, the Paradox Integration layer records the contradiction, contextualizes both, and treats the paradox itself as a higher-order Principle.

## Acceptance Criteria

- Contradiction detection is automatic whenever a new Principle cites evidence that disagrees with an existing one.
- Paradox nodes are first-class in the Knowledge Graph with a `RESOLVES_IN_REGIME` relationship to regime nodes.
- Signal Blender applies regime-aware weighting that respects paradox resolutions.
