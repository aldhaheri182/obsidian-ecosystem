# Layer 11 — Noosphere

**Core innovation:** Emergent collective self-awareness.

A continuously-updated graph that represents the ecosystem's own aggregate beliefs — what it collectively "believes" about each instrument, regime, and strategy. The Noosphere is maintained by Mnemosyne; agents may query it to calibrate their own confidence.

## Acceptance Criteria

- Noosphere state is a signed, time-stamped snapshot; snapshots form a first-class stream in Mnemosyne.
- Disagreement between agent private beliefs and the Noosphere is tracked and exposed in the UI.
