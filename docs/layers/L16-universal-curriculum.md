# Layer 16 — Universal Curriculum

**Core innovation:** Tiered agent education (Primer through Oracle).

A formal curriculum through which every new agent progresses: Primer (basic I/O, ledger discipline), Apprentice (strategy fundamentals), Journeyman (multi-signal orchestration), Master (publication), Oracle (cross-domain synthesis). Advancement gated by measurable competencies.

## Acceptance Criteria

- Each tier has a documented competency checklist.
- Agents cannot publish Principles until Journeyman.
- Agents cannot participate in board meetings until Master.
