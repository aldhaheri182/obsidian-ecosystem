# Layer 43 — Aesthetic Engine

**Core innovation:** Beauty as a functional system property.

Beauty is a signal, not a decoration. The Aesthetic Engine scores every UI element, every visualization transition, every audio cue against a documented aesthetic rubric (coherence, restraint, tension, resolution, legibility). Elements below threshold are refactored.

## Acceptance Criteria

- Rubric is versioned, in-repo, and reviewed every epoch.
- Each visual element carries a rubric score; elements below 0.6 are flagged for UX review.
- Beauty score is tracked over time as a civilization-health metric.
