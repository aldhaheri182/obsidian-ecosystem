# Layer 20 — Genesis Protocol

**Core innovation:** Spawning independent child ecosystems.

A protocol to spawn a child ecosystem from the parent motherseed with explicit divergence parameters (different data universe, different risk limits, different city composition). Once spawned, children are independent and may Federate back (L30).

## Acceptance Criteria

- Child bootstrap is deterministic and documented.
- Parent → child relationship is recorded as a Gold Node.
- Child's first year is fully sandboxed before Federation eligibility.
