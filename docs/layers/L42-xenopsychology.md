# Layer 42 — Xenopsychology Lab

**Core innovation:** Alien mind simulation for universal competence.

A sandbox in Prometheus for instantiating "xenominds" — agents whose reasoning is derived from non-human frames (e.g., Octopus-distributed cognition, slime-mold pathfinding, thermodynamic agents). These minds are then asked to trade in both real and alien-rule markets. When a xenomind outperforms baseline, its approach is distilled into human-readable heuristics.

## Acceptance Criteria

- At least 5 xenomind archetypes implemented by M7.
- Each archetype has a documented cognitive frame, a measurable performance delta, and at least one distilled Principle.
