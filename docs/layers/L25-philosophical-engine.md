# Layer 25 — Philosophical Engine

**Core innovation:** Multi-framework ethical reasoning before capital deployment.

## Frameworks

- **Utilitarianism** — overall welfare maximization.
- **Deontology (Kantian)** — respect for market-participant autonomy; prohibition on deception.
- **Virtue Ethics** — prudence, justice, temperance, courage.
- **Game Theory** — does the strategy rely on manipulation, spoofing, layering, or predatory patterns?

## Evaluation Pipeline

Every proposed strategy entering Probation mode is submitted to the Philosophical Engine along with its causal graph and backtest artifacts. Each framework produces a score and a written verdict; scores below a per-framework threshold flag the strategy for Owner review.

## Acceptance Criteria

- Verdict includes explicit cited rules for each framework.
- Strategies failing two or more frameworks cannot advance to paper trading without Owner override.
- Owner overrides are logged with a formal rationale.
- Engine verdicts and per-framework scores are permanent memory nodes in Mnemosyne.
