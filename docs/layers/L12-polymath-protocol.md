# Layer 12 — Polymath Protocol

**Core innovation:** Universal intelligence platform for non-financial domains.

The Obsidian runtime (NATS, ledger, supervisor, city metaphor) is domain-agnostic. Polymath Protocol documents how to instantiate the platform for a new domain (scientific research, logistics optimization, supply chain) by swapping Aletheia for a domain-specific Productivity City.

## Acceptance Criteria

- A reference port exists (e.g., a mini Polymath city for paper-review pipelines).
- Contract between domain-city and ecosystem-core is documented.
- A domain port passes its own acceptance suite analogous to M0.
