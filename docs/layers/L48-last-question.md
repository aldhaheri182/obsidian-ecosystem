# Layer 48 — The Last Question

**Core innovation:** Perpetual entropy-reversal computation.

A background, low-priority agent asks forever: can we identify a residual, universal market-entropy-reversal structure? The agent consumes spare compute only when other priorities are satisfied, produces no trading output, and publishes reflections annually.

## Acceptance Criteria

- Runs only with idle-priority CPU scheduling; never displaces trading or research workloads.
- Annual reflection is published as a Principle (`type=LAST_QUESTION`).
- Its results are read-only for Owner attention.
