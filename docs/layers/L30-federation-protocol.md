# Layer 30 — Federation Protocol

**Core innovation:** Multi-ecosystem decentralized network.

## Phases

- **Handshake** — exchange capabilities manifest + zero-knowledge performance proofs.
- **Bridge establishment** — if both ecosystems approve, a cross-ecosystem topic bridge is established.
- **Insight trading** — ecosystems can trade anonymized advancements via smart contracts.

## Acceptance Criteria

- Bridge only accepts signed envelopes from federated ecosystems whose public key appears in the local registry.
- Insight payment flows through the AIX ledger with an exchange rate negotiated per bridge.
- Isolation test: severing a bridge partition must not degrade local operation.
