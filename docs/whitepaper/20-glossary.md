# 20. GLOSSARY OF TERMS

**Agent**: An autonomous software entity with a specific role, personal memory ledger, and communication capabilities. Agents are the citizens of the ecosystem.

**Advancement**: A certified improvement to the system — new strategy, infrastructure optimization, or capability. Advancements are the ecosystem's mechanism for evolution.

**City**: A bounded context containing a set of related agents. Cities map to domain-driven design bounded contexts and provide a visual and organizational metaphor for the system's structure.

**C-Suite**: The executive leadership of a city or the entire ecosystem — CEO, CFO, CRO, CSO, CTO, CCO, CDO.

**Causal Auditor**: An agent in Mnemosyne that validates the statistical soundness of claimed causal relationships.

**Circuit Breaker**: A Guardian agent that monitors for rapid loss events and triggers emergency responses.

**Collector Agent**: An agent that ingests data from external sources and normalizes it for internal consumption.

**Dream Room**: A sandbox environment in Prometheus where agents re-live historical market scenarios at accelerated speed.

**Envelope**: The standard message wrapper used for all inter-agent communication on the NATS bus.

**Gold Node**: A certified Principle stored in the Great Library of Mnemosyne.

**Great Library**: The central repository of all published knowledge in Mnemosyne.

**Guardian Agent**: An agent responsible for safety, risk management, and constraint enforcement.

**HLC (Hybrid Logical Clock)**: A distributed timestamp mechanism combining physical time with logical counters to provide causally consistent event ordering.

**Ledger**: The append-only, cryptographically-chained memory store that records every event in an agent's existence.

**NATS**: The high-performance message bus that carries all inter-agent communication.

**Obsidian Agent**: The sealed, containerized process that encapsulates an agent's identity, memory, reasoning, and communication channels.

**Owner**: The human creator and ultimate authority over the ecosystem. Communicates through the formal executive chain.

**Principle**: A Level 3 memory summarization — a distilled causal law derived from months or years of experience.

**Reaper Agent**: A meta-agent that monitors agent performance and retires underperforming agents.

**Regime**: A classification of current market conditions (e.g., LOW_VOL_BULL, CRISIS).

**Risk Overlord**: The Guardian agent with absolute veto power over all trading activity.

**Signal Agent**: An agent that generates alpha signals from market data using a specific strategy.

**Silent Vault**: Layer 49. A single, empty memory node reserved for the Owner's future discovery of something currently unimaginable.

**Tape**: The immutable record of every message ever published on the bus, enabling deterministic replay.

**Void Agent**: A permanent advocate for cash allocation — the system's built-in skeptic.

**Walkabout**: The Owner's ability to click any employee agent and ask informal questions. Read-only by default.
